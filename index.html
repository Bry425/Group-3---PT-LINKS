Introduction

In HTML, the <a> tag (also called the anchor tag) is used for creating link in html. It allows you to link websites, other pages in your project, specific sections of the same page, email address, phone number, and even files for download. Understanding how it works is essential because links can open a lot of ways to be much more creative in your web page.

How To Properly Use

To put a link somewhere in your webpage, you would first need to know how to properly use the anchor tag. The basic form of an anchor tag is:
<a href="URL OR FILE LOCATION">Clickable text that the reader will see</a> You can also link to external websites, other pages within your project, specific sections of the same page, email address, or phone number.

Types of URLs

There is two types of URL that you can use in the anchor tag, Absolute and Relative URL.

What is a Absolute URL

An "Absolute URL" is a full web address that points directly to a specific location on the internet, including the protocol(The http:// or https://), the domain name, and optionally the path to a file or page.

Example: <a href="https://www.google.com/">Google</a>

What is a Relative URL

An Relative URL" is a file location that is within your own project folder rather than specifying the full web address, thats why its called a "Relative URL", because its on the same project folder.

Example: <a href="about.html">About</a>

Target Attribute

By default, the linked page will be displayed in the current browser window. To change this, you must specify another target for the link.

The target atrribute specifies where to open the linked document or page

The target attribute can have one of the following values:

_self - The Default. Opens the page or document in the same window/tab as it was clicked
_blank - Opens the document in a new window or tab
_parent - Opens the document in the parent frame
_top - Opens the document in the full body of the window

Example: <a href="https:www.google.com" target="_blank">Google</a>

Other Uses of Links

Using an Image

To use an image as a link, just put the <img> tag inside the <a> tag: <a href="https://example.com"> <img src="example.jpg" alt="Description"></a>

Link to an Email

Use "mailto:" inside the href attribute to create a link that opens designated email. Example: <a href="mailto:example@gmail.com">Email</a>

Using a Button as a Link

JavaScript allows you to specify what happens at certain events, so we can use it to change what happens after clicking a button. Example: <button onclick="document.location='default.asp'">Example</button>

Linking to a specific part of a page

The link: <a href="#section2">Section 2</a> The Specific Part: <h2 id=section2">Section 2</h2> or you can do it like this: The link: <a href="#contact">Jump to Contact section</a> The specific part: <section id="contact"> <h2>Contact Us</h2> <p>Email: example@gmail.com</p> </section>

Conlusion

There is still many more ways to utilize links in html but we went through the most basic and simple ones, and essentially, being able to utilize Links or the use of anchor tag can make your web page much better, such as allowing users to go to specific part of a page just by clicking the link or redirecting reader to a reference you want them to see and so on. Overall, understanding how to properly use links can make reader of your webpage have a smoother navigation experience and you'll also be able to connect information much easier.
